# Installation

Choose one ``Engine.ini`` config and other files if needed

<details>
<summary>Optional files in Common folder (Click to expand)</summary>

---

``DeviceProfiles.ini`` to show Lumen in game settings

``Game.ini`` to skip first two intro videos

``Input.ini`` to disable mouse smoothing and FoV scaling

---

</details>

Copy the file(s) to this location:

> %LocalAppData%\HT\Saved_Global\Config\Windows

macOS users should copy them to this location instead:

> ~/Library/Containers/com.hottagames.nte/Data/Library/Application Support/Epic/HT/Saved/Config/Mac/ <br>
> <sub>To go to this location on macOS, open Finder, type <kbd>⌘ Cmd</kbd> + <kbd>⇧ Shift</kbd> + <kbd>G</kbd>, and paste the path above.</sub>

> [!IMPORTANT]
> Set the files to read-only for ``DeviceProfiles.ini``, ``Engine.ini``, ``Game.ini`` and ``Input.ini`` or it'll get deleted.
> 
> Windows: right click the file, select Properties, then check "Read-only" in the window that appears.<br>
> macOS: right click the file, select Get Info, then check "Locked" in the window that appears.
